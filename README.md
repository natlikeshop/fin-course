# 🎓 Fin-Course - Premium Financial E-Learning Platform

ระบบแพลตฟอร์มจัดการเรียนรู้ออนไลน์ด้านการเงินระดับพรีเมียม (Fin-Course) ที่ถูกพัฒนาขึ้นด้วย **PHP Native (PDO)**, **MySQL** และ **Vanilla CSS** ที่เน้นความสวยงามในรูปแบบ **Sleek Light-Glassmorphism Design** สะอาดตา ใช้งานง่าย และมีความยืดหยุ่นสูง (Responsive Design) รองรับอุปกรณ์ทุกรูปแบบ

---

## 🎨 แนวคิดการออกแบบและสถาปัตยกรรม (Design & Architecture)

1. **Sleek Light-Glassmorphism Theme**: การอัปเกรดโทนสีสว่างสุดโมเดิร์น แบล็กกราวด์เฉดสี Slate-Blue จาง ๆ (`#f5f7fb`) ผสมผสานกับการ์ดกระจกโปร่งแสง (Glassmorphic Cards) เล่นระดับมิติและเงาที่นุ่มนวล
2. **Centralized Routing**: การสลับหน้าผ่านตัวจัดการเส้นทางศูนย์กลาง `index.php?page=...` เพื่อควบคุมความปลอดภัยของระบบและการเข้าถึงได้อย่างมีประสิทธิภาพ
3. **Clean Modular Structure**: แยกส่วนการทำงานของไฟล์อย่างชัดเจนตามสิทธิ์การใช้งาน (Role-based Directories) ป้องกันความทับซ้อนของโค้ด

---

## 🚀 ฟีเจอร์แยกตามบทบาทผู้ใช้งาน (Role-based Features)

### 🧑‍🎓 1. ระบบสำหรับนักเรียน (Student Dashboard)
* **หน้าลงทะเบียน & สมัครเรียนออนไลน์ (Enrollment System)**: สมัครเรียนวิชาการเงินที่สนใจได้ฟรีด้วยคลิกเดียว
* **เครื่องเล่นบทเรียนอินเตอร์แอกทีฟ (Lesson Player)**: หน้าศึกษาบทเรียนที่มีทั้งคลิปวิดีโอ เนื้อหาประกอบ และระบบดาวน์โหลดไฟล์สรุปเอกสาร (PDF)
* **สารบัญบทเรียนด่วน (Sidebar Navigator)**: เลือกสลับบทเรียนในวิชาเรียนได้อย่างรวดเร็ว
* **ระบบทำแบบทดสอบหลังเรียน (Quiz System)**: วัดความรู้ทางการเงินที่เพิ่งเรียนจบ พร้อมวิเคราะห์ตรวจคะแนนทันทีหลังทำเสร็จ
* **หน้าโปรไฟล์และประวัติการเรียน (Student Profile)**: รายงานสรุปคะแนนประวัติการสอบทุกวิชาในรูปแบบตารางและป้ายสถานะผ่าน/ไม่ผ่านที่สวยงาม

### 👩‍🏫 2. ระบบสำหรับผู้สอน (Teacher Control Panel)
* **ระบบจัดการวิชาเรียนส่วนตัว (Course Management)**:
  * หน้าจัดการรายวิชาในรูปแบบเรียบร้อย สบายตา
  * หน้าสร้างวิชาเรียนใหม่แบบแยกเฉพาะส่วน (Dedicated Create Screen) ลดความซับซ้อนของหน้าหลัก
* **ระบบการจัดการปุ่มแบบเรียบร้อย (Organized Buttons)**: จัดกลุ่มปุ่มงานปกติ (บทเรียน/ข้อสอบ) เป็นชิ้นเดียวกัน และแยกปุ่มลบอันตรายด้วยรูปแบบโปร่งใส (Outlined Red Button) เพื่อการทำงานที่รวดเร็วและปลอดภัย
* **การจัดการบทเรียนภายในวิชา (Lesson Management)**: เพิ่ม แก้ไข และลบบทเรียน กำหนดลิงก์วิดีโอ และอัปโหลดไฟล์ PDF
* **การจัดการคลังแบบทดสอบ (Quiz Management)**: ตั้งโจทย์คำถามและเฉลยคำตอบสำหรับแบบทดสอบย่อยของวิชานั้น ๆ
* **รายงานสถิติคะแนนนักเรียน (Student Reports)**: ตารางติดตามคะแนนของนักเรียนทุกคนที่เข้ามาศึกษาในรายวิชา

### 🔑 3. ระบบสำหรับผู้ดูแลระบบ (Admin Console)
* **แผงควบคุมสถิติภาพรวม (Admin Dashboard - Overview)**: ตรวจดูสถิติจำนวนผู้ใช้ รายวิชา บทเรียน และประวัติคะแนนที่สอบผ่าน/ไม่ผ่านทั้งหมดในระบบ
* **ระบบจัดการสิทธิ์ผู้ใช้งาน (User Account Management)**: เพิ่มบัญชีใหม่ แก้ไขชื่อ-นามสกุล เปลี่ยนบทบาท (Admin/Teacher/Student) และลบผู้ใช้ที่ไม่มีสิทธิ์ใช้งาน
* **ระบบดูแลความสงบเรียบร้อย (Course Moderation)**: ตรวจเช็กและดูแลรายวิชาทั้งหมดในระบบ

└── index.php              # ตัวสลับหน้าเราเตอร์ศูนย์กลาง (Centralized Router)

## ⚙️ วิธีการติดตั้งและตั้งค่าโปรเจกต์ (Installation Guide)

1. **คัดลอกไฟล์**: นำโฟลเดอร์ `fin-course` ไปวางในตำแหน่งเว็บรูทของเซิร์ฟเวอร์จำลองของคุณ (สำหรับ XAMPP คือ `C:\xampp\htdocs\fin-course`)
2. **สร้างฐานข้อมูล**: 
   * เปิด **MySQL** ผ่านโปรแกรมควบคุม (เช่น XAMPP Control Panel)
   * เข้าสู่หน้าจัดการฐานข้อมูล [http://localhost/phpmyadmin](http://localhost/phpmyadmin)
   * สร้างฐานข้อมูลใหม่ขึ้นมา ตั้งชื่อว่า `fin_course`
3. **นำเข้าโครงสร้าง SQL**:
   * คลิกที่ฐานข้อมูล `fin_course` ที่สร้างขึ้น
   * ไปที่แท็บ **Import**
   * เลือกไฟล์ข้อมูล `.sql` ที่แนบมากับระบบ แล้วกดยืนยันเพื่อจัดเตรียมตารางข้อมูลให้ครบถ้วน
4. **ตั้งค่าฐานข้อมูล**:
   * เปิดไฟล์ `config.php`
   * แก้ไขตัวแปรฐานข้อมูล (เช่น Username, Password และชื่อ Database) ให้ตรงกับระบบจำลองของคุณ
5. **เริ่มต้นใช้งาน**: เข้าสู่ระบบผ่านเว็บเบราว์เซอร์ทาง: [http://localhost/fin-course](http://localhost/fin-course)
